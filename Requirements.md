ENFU ANALYTICS — REQUIREMENTS (INITIAL)

GOAL:
Mobile app that lets users connect a social media account and see analytics about their exposure, engagement, and growth.

KEY FEATURES:
1. Login with social media (start with Instagram or Twitter/X).
2. Fetch engagement data (followers, likes, comments, reach).
3. Show a dashboard with:
   • Daily follower growth
   • Engagement rate
   • Top performing posts
4. AI-generated insights:
   • Best posting time
   • Content suggestions
   • Audience analysis
5. Save user history for trend tracking.

PLATFORM:
Mobile (initial release).

BACKEND:
Python (FastAPI), PostgreSQL or Firebase.

FRONTEND:
React Native or Flutter (decide later).

PRIVACY:
User data stored securely, no public access.

NEXT STEPS:
1. Define database structure.
2. Create minimal API endpoints.
3. Choose first platform (Instagram or Twitter/X).
